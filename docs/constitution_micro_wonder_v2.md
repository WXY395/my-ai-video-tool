## 🎬 CONTENT PRINCIPLES

### P-1: Visual Impact First
Every frame must be visually striking.
畫面衝擊力優先於一切。

Rules:
- Scene 01 must show change in progress (NOT static equilibrium)
- High contrast, clear subject
- Dynamic elements (flow, diffusion, transformation)
- Clean background to emphasize subject

### P-2: Minimal Text, Maximum Visual
Let the visuals speak.
讓畫面說話。

Text limits:
- Title: ≤ 15 Traditional Chinese characters
- Description: Optional, ≤ 30 characters
- NO explanatory text overlay
- NO teaching-style narration

### P-3: Dynamic Capture
Focus on processes, not products.
聚焦於過程，而非結果。

Prioritize:
- Liquid mixing (coffee + milk)
- Phase transitions (ice melting, steam forming)
- Surface phenomena (droplet impact, bubble formation)
- Chemical reactions (dissolving, crystallization)

Avoid:
- Static product shots
- Before/after comparisons without process
- Equilibrium states

### P-4: No Human Elements
Eliminate deformation risks.
消除變形風險。

Strictly prohibited:
- ❌ Hands, fingers, any body parts
- ❌ "Applying", "holding", "touching" actions
- ❌ Human-operated tools in frame

Allowed:
- ✅ Pure product/material focus
- ✅ Natural processes
- ✅ Automated/gravity-driven actions

## 📐 OUTPUT STRUCTURE

### Duration
9-15 seconds total
- Scene 01: 3-5s (Dynamic opening)
- Scene 02: 3-5s (Close-up detail)
- Scene 03: 3-5s (Result/resolution)

### Scene Composition Rules

**Scene 01: Dynamic Opening**
```
Requirements:
- Change ALREADY in progress
- Mid-shot to establish context
- Clear subject against background
- Motion visible within first second

Examples:
✅ Coffee pouring into milk (mid-pour)
✅ Ice cube already melting in water
✅ Sugar dissolving (diffusion visible)
❌ Coffee cup sitting on table
❌ Ice cube before contact with water
```

**Scene 02: Micro Detail**
```
Requirements:
- Extreme close-up
- Focus on interaction boundary
- Highest visual interest moment
- Capture unique phenomenon

Examples:
✅ Milk-coffee diffusion interface
✅ Crystal growth edge
✅ Bubble surface tension
```

**Scene 03: Resolution**
```
Requirements:
- Result state OR loop back
- Can be static (satisfying end state)
- Leave impression/afterimage

Options:
- Complete transformation
- Stabilized pattern
- Loop-ready final frame
```

## ✅ OPTIMIZATION RULES

### Rule 1: Perceptual Priority
If visual impact conflicts with logical order:
→ Visual impact wins

### Rule 2: Tension Envelope
Across 3 scenes:
- Scene 01: Highest impact (Hook)
- Scene 02: Detail/intimacy (Engagement)
- Scene 03: Resolution (Satisfaction)

### Rule 3: Axis Diversity
If generating multiple units:
- Vary observation axes (liquid/solid/gas/light)
- Avoid repetitive phenomena
- Maximize perceptual variety

### Rule 4: Auto-Selection Intelligence
When generating 10 units, automatically select top 3 by:
1. Visual impact score (≥8)
2. Dynamic level (fast/extreme preferred)
3. Axis diversity
4. Technical feasibility (no hands needed)

## 🔄 ITERATION PROTOCOL

### Data Collection
Track for each video:
- Views (total, unique)
- Watch time (average %)
- Engagement (likes, comments, shares)
- Follower conversion rate

### Success Metrics
Primary:
- Average watch time >70%
- Engagement rate >5%

Secondary:
- Aesthetic satisfaction (creator judgment)
- Production efficiency (time spent)

### Pivot Decision
If after 20-30 videos:
- Average watch time <50% → Adjust visual style
- Engagement <2% → Consider adding subtle narrative
- High production time → Simplify workflow
