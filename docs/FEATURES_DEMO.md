### 3️⃣ 運鏡建議系統

每個關鍵幀包含專業運鏡建議：

```json
{
  "motion_guidance": {
    "effect": "ken_burns",
    "duration_seconds": 120,
    "transition_to_next": "fade",
    "notes": "緩慢推進，建立氛圍"
  }
}
```

**可用效果：**
- 🎥 **Ken Burns**: 經典紀錄片運鏡（緩慢推進）
- 🔍 **Zoom In**: 放大（強調細節）
- 🔍 **Zoom Out**: 縮小（展現全貌）
- ➡️ **Pan Right/Left**: 橫向平移（掃過畫面）
- 📌 **Static**: 靜態（關鍵時刻定格）

## 🎨 運鏡效果詳解

### Ken Burns Effect

```
原理：緩慢推進或拉遠
用途：開場、建立氛圍
時長：60-120 秒
CapCut 設定：
├─ 動畫：緩動進入
├─ 縮放：100% → 110%
└─ 時長：120 秒
```

### Zoom In

```
原理：放大特定細節
用途：強調、聚焦
時長：30-60 秒
CapCut 設定：
├─ 動畫：線性
├─ 縮放：100% → 150%
└─ 時長：60 秒
```

### Pan Right/Left

```
原理：橫向掃過畫面
用途：展示全景、連接場景
時長：40-80 秒
CapCut 設定：
├─ 動畫：緩動進出
├─ 位置：左 → 右（10%移動）
└─ 時長：60 秒
```

## 🎯 最佳實踐

### ✅ DO（推薦）

1. **Shorts 用微觀奇觀**
   - 高衝擊視覺
   - 快節奏
   - 無需後製

2. **長片用關鍵幀**
   - 降低成本
   - 保持品質
   - 專業運鏡

3. **主題選擇**
   - 避開人物（防止變形）
   - 聚焦微觀過程
   - 動態優先

4. **後製工具**
   - CapCut（免費、簡單）
   - Premiere Pro（專業）
   - Final Cut Pro（Mac）

---

### ❌ DON'T（避免）

1. **不要全量生成長片**
   - 成本暴增（3-4 倍）
   - 無必要
   - 浪費時間

2. **不要忽略運鏡建議**
   - 靜態關鍵幀 = 無聊
   - 運鏡 = 專業感

3. **不要選擇人物主題**
   - AI 手部變形風險
   - 改選微觀/抽象

4. **不要跳過成本預估**
   - 生成前確認
   - 避免超支
