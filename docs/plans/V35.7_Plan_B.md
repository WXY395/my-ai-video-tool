## Section 2: Module Design

### bridge_adapter.ts
- Re-exports `generateImage`, `renderContentPack`, `generateSSOT` from `./services/gemini`
- Locks `ASPECT_RATIO = "9:16"` and `IMAGE_MODE = GenerationMode.EXPLAIN`
- `generateImageOfficial(prompt)` → converts base64 → `Buffer`
- No business logic, no Director calls

### DirectorService.ts (Python Director 1:1 port)
| Python | TypeScript | Logic |
|--------|-----------|-------|
| `segment_script()` | `segmentScript()` | Split on `[。，、！？；…\n]+`, max 10 chars/segment |
| `align_scenes()` | `alignScenes()` | Pad with Auto_Segment if scenes < VO; trim if > |
| `needs_veo()` | `needsVeo()` | scene_idx===0 OR last OR VEO_TRIGGER_VERBS match |
| `sfx_for_scene()` | `sfxForScene()` | Hook/Payoff→transition, humor markers→humor, else ambient |
| `ms_to_smpte()` | `msToSmpte()` | `Frames = round(ms/1000*30)`; HH:MM:SS:FF |
| `extract_metaphor()` | `extractMetaphor()` | 14-entry Map lookup |
| `static_action()` | `staticAction()` | "Camera slow zoom in (1.2×) \| Ken Burns effect" |

Duration formula: `0.3 + voText.length * 0.19 + 0.5` seconds

### cli.ts flow
1. `process.argv[2]` → topic
2. `generateSSOTOfficial(topic)` → SSOT JSON
3. `renderPackOfficial(ssot)` → ContentPack `{ vo_lines, image_prompts, seo_txt, cover_prompt, ... }`
4. `DirectorService.alignScenes(voLines, imagePrompts, topic)` → aligned scenes[]
5. Cover: `generateImageOfficial(cover_prompt)` → `cover.png`
6. scene_001: `generateImageOfficial(scene.prompt)` → `scene_001.png` (Nano Banana via gemini.ts)
7. scene_002+: Replicate flux-schnell → `scene_NNN.png`
8. `buildRunbook()` → `runbook.md`
9. `buildSrt(voLines)` + `buildSubtitlesSrt()` → `draft_vo.srt`, `subtitles.srt`
10. `cleanSeoTxt(seo_txt)` → `seo.txt`
11. `buildManifest(saved)` → `final_manifest.json`
12. `fs.mkdirSync(outDir)` + all `fs.writeFileSync` calls

### Output spec (identical to Python)
`cover.png`, `scene_001..N.png`, `subtitles.srt`, `draft_vo.srt`,
`runbook.md`, `seo.txt`, `final_manifest.json`

## Section 3: Error Handling + Flux + Installation

### Error handling
- **429**: exponential backoff `2^n * 2000ms`, max 3 retries (mirrors gemini.ts existing logic)
- **Image failure**: DALL-E 3 fallback via OpenAI SDK (same as Python)
- **JSON parse failure**: throw named error, halt pipeline

### Flux-schnell (scene_002+)
Direct Replicate call in `cli.ts` (not through bridge_adapter):
```typescript
import Replicate from "replicate";
const output = await replicate.run("black-forest-labs/flux-schnell", {
  input: { prompt: applyStyleFluxScene(base), aspect_ratio: "9:16",
           output_format: "png", num_outputs: 1 }
});
```
Style: `applyStyleFluxScene()` — double no-text barrier + minimalist ink splash abstraction
(mirrors Python `_FLUX_NO_TEXT_PREFIX` + `_FLUX_STYLE_CORE`)

### Installation (minimal footprint)
```bash
cd bridge/official_core
npm install --save-dev tsx replicate openai
```

### YAGNI (explicitly excluded)

- No changes to `tsconfig.json`
