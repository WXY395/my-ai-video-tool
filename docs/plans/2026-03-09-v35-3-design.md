# V35.3 Audio-Visual Factory — Design Doc
Date: 2026-03-09

## Changes from V35.2

### Schema
- `ImagePrompt`: + `veo_prompt: str` (三元結構)
- Gemini 輸出 `vo_lines/subtitle_lines` 為純文字列表，SRT 本地生成

### Config DNA
- `STYLE_TEMPLATE = "{prompt}\n\nFauvism..."` — {prompt} 嵌入點
- `IMAGE_ASPECT_RATIO = "9:16"`, `IMAGE_SIZE = "1K"` (aspirational)
- `apply_style_template(prompt, render_text=None)` — [TEXT] 指令開關

### SRT 標準化
- `generate_srt(lines: list[str]) -> str`
- 每段：序號 + 時間軸 + 內容 + `\n`
- 時碼：`0.3 + chars×0.19 + 0.5`，段間 gap 0.1s
