# V35.2 Audio-Visual Factory — Design Doc
Date: 2026-03-08

## Changes from V35.0

### 1. Config DNA (hardcoded constants)
- `DIRECTOR_SYSTEM_INSTRUCTION` — 百萬 YouTuber 影音導演角色設定
- `STYLE_SUFFIX` — Fauvism + Chinese ink painting 風格後綴，附加至所有圖像 prompt
- `compute_srt_duration(text)` — 時碼算法：`0.3 + len(text) * 0.19 + 0.5`

### 2. Schema Updates
- `ContentPack` 新增 `runbook_all_in_one: str`
  - **定義**：Markdown 格式製作指南，含 Timeline Tracks + 結構化時碼
  - **來源**：本地 `build_runbook()` 函數建構，非 Gemini 輸出
- `ImagePrompt` 維持 `chapter, prompt, filename`

### 3. Two-Phase Knowledge Mounting
- Phase 1 (可選)：讀取 `--ssot` 文字檔 → `gemini-3-flash-preview` 提取結構化事實
- Phase 2：`gemini-3.1-pro-preview` + `DIRECTOR_SYSTEM_INSTRUCTION` → 渲染 `ContentPack`
- NotebookLM 整合：由使用者先用 `notebooklm` 技能查詢並儲存為 `ssot.txt`，再傳入 `--ssot` 旗標

### 4. Visual Style Filter
- 所有圖像 prompt（cover + scenes）在送出前自動附加 `STYLE_SUFFIX`
- 比例強制 `9:16`（Nano Banana 2 + Flux 兩者）

### 5. runbook_all_in_one 結構
```
# {topic} — 製作指南 (V35.2)
## Timeline Tracks
| # | IN → OUT | Chapter | VO | 字幕 | 分鏡 |
|---|----------|---------|-----|------|------|
...
## 完整旁白腳本 / 精簡字幕 / SEO 文案 / 分鏡列表
```
時碼由 `compute_srt_duration()` 逐段累加。

### 6. File Output
新增 `runbook.md` 至輸出目錄。
